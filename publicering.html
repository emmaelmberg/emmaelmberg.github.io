Netlify

Jag har använt Netlify för att publicera denna laboration genom att koppla upp mitt GitHub-konto till tjänsten. Därefter skapade jag ett nytt repo i GitHub som jag valde att distribuera i Netlify. Efter varje commit gör jag en push som uppdaterar repot och webbplatsen på Netlify.

Fördelar

  • Kostnadsfritt upp till 100GB brandbredd/månad
  • Enkelt att komma igång
  • Går att koppla till GitHub repo
