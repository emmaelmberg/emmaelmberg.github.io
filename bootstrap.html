Bootstrap

Om ramverket

Bootstrap är ett CSS-ramverk med öppen källkod som används för att skapa responsiva webbplatser med förbestämda breakpoints [1]. Här används containers i kombination med ett standardiserat grid-system och flexbox. Det finns ett antal olika kulörer att välja mellan för att styla bland annat textfärg, bakgrundsfärg och knappar.

Ramverket skapades av Twitter och blev open-sourced 2011 och är en av de mest populära CSS-ramverken för att styla webbplatser [2]. Ramverket går att använda med enbart CSS eller med CSS i kombination med JavaScript. För att börja använda Bootstrap går det att ladda ner, installera via NPM, eller CDN genom att placera länk till stylesheet och JavaScript i HTML-dokumentet. I skrivande stund är den senaste versionen 5.3.1.

Intryck

Jag tycker om att ramverket är enkelt att lära sig och det går relativt fort att göra en enkel design som är responsiv, speciellt när det gäller menyer. Det underlättar även designen av tabeller och formulär som oftast slukar en hel del tid att styla med CSS. Jag gillar även att det finns flera olika sätt att börja använda ramverket och det går att sätta igång att koda direkt med CDN, som jag testade i det här momentet.

Det som jag tycker är mindre bra med ramverket är att inte ha lika stor kontroll över bland annat typsnitt och kulörer. Efter att ha utvecklat med ramverket så upplever jag att det redan finns en tydlig förbestämd stil som är enkel att känna igen. De flesta webbplatser som använder ramverket ser ganska lika ut, vilket känns lite tråkigt.
